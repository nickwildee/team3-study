# Weekly Branch – 학습 공유 & PR 리뷰 공간

이 브랜치는 조원들이 매일 학습한 내용을 공유하고, 리뷰를 통해 함께 성장하기 위한 공간입니다.

## 📌 PR 작성 규칙

### ✅ 제목

- `[08-05] 조원이름`  
  예시: `[08-05] 혁`

### ✅ 내용

- 오늘 학습한 내용 요약
- 코드 설명 또는 질문
- 피드백 받고 싶은 부분

### ✅ 머지 조건

- 팀원 전원의 Approve
- Squash and Merge 사용

### ✅ 머지 시 커밋 메시지 작성 규칙

PR을 `Squash and Merge`할 때 최종 커밋 메시지는 다음 규칙을 따릅니다:

- 형식: `[날짜] 조원이름 - 학습 정리`
- 예시: `[08-05] 혁 - 학습 정리`
