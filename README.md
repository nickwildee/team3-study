# 3팀 학습 내용 공유 저장소

## 목적

- 학습 내용 공유
- 서로의 코드 리뷰

## 브랜치 구조

- main: 한 주 단위 정리본
- weekly : PR 리뷰 및 머지 대상
- 각자 이름 브랜치 : 개인 작업용

## 디렉토리 구조

```js
main
  └── sungWoo/ //본인 이름으로 된 디렉토리

              └── index.html // 해당일자 작업물
              └── style.css
              └── ...
```
